# バンコク電線解析メガネ (Web Server v2) - 構築・利用ガイド

このプロジェクトは、バンコクの複雑な電線を解析し、AI（VLM）の知見を画像上に可視化するWebサービスです。
v2では、スマホのブラウザから直接カメラを起動し、撮影・アップロードして解析結果を確認できる機能が追加されました。

## 1. 構成ファイル
- `final_server_v2.py`: メインプログラム（FastAPI + Gemini API）
- `persona.txt`: AIエージェントの性格・役割を定義するファイル
- `research_data.md`: 電線の識別ルールなどの技術ナレッジを記したファイル

## 2. 準備

### 必要なライブラリのインストール
```bash
pip install fastapi uvicorn google-genai Pillow python-multipart

```

### APIキーの設定

環境変数にGemini APIのキーを設定します。

```bash
# Mac/Linux
export GEMINI_API_KEY="あなたのAPIキー"

# Windows (コマンドプロンプト)
set GEMINI_API_KEY="あなたのAPIキー"

```

## 3. サーバーの起動

```bash
python final_server_v3.py

```

デフォルトで `http://0.0.0.0:8000` で待機します。

## 4. 利用方法

### PCから利用する場合

ブラウザで `http://localhost:8000` にアクセスしてください。

### スマホから利用する場合（現場での利用を想定）

1. PCとスマホを同じWi-Fiネットワークに接続します。
2. PCのローカルIPアドレス（例: `192.168.1.10`）を調べます。
3. スマホのブラウザから `http://192.168.1.10:8000` にアクセスします。
4. 「ファイルを選択」をタップし、カメラを起動して電線を撮影します。
5. 「解析開始」をタップすると、数秒後に注釈付きの画像が表示されます。

## 5. カスタマイズ

* **AIの振る舞いを変える**: `persona.txt` の内容を編集してください。
* **識別知識を更新する**: `research_data.md` に新しい識別ルール（新しい事業者の色など）を追記してください。
* **フォントの調整**: `final_server_v2.py` 内のフォントパス設定（`font_path`）を、実行環境に合わせて調整してください。
